Cyberpunk 2077

dit is alleen de home page waar ik vertel waarom ik mijn eindopdracht maak over deze game. De reden is omdat dit de enige game is waar ik een bijna 60 uur safe heb en nog steeds veel meer kan doen. Voor alle shit die de game kreeg door de slechte release (en die was erg door alle bugs in de game) is het nu echt een goede game geworden. Ik vind het nu een geweldige game en ik wist ook nog niet waar mijn eindopdracht over wouw maken en ik starte de game weer op en dacht ja hier ga ik het over maken. je kan in de menu balk meer info vinden over de game bij de over de game optie.
